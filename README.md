## Getting Started
To get a local copy up and running follow these simple steps.

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/peatear-anthony/abenomask.git
```
2. Install  packages
```sh
pip install -r requirments.txt
```
3. Run App
```sh
python run.py
```
